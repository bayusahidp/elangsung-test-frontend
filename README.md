## Project setup
Menggunakan vue@cli 2.6.11
- buka terminal / cmd pada PC / Laptop anda
- tentukan lokasi folder
- git clone {{ url repo ini }}
- cd {{ nama repo / folder project }}
- npm run serve

aplikasi berjalan pada : "http://localhost:8080"
